# Credit Card Fraud Detection & Risk Analysis - SQL Project

## Project Overview

This project focuses on analyzing credit card transaction data using SQL to identify fraud patterns, risky customer behavior, and suspicious transactions.

The project demonstrates:

- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Fraud trend analysis
- Risk segmentation
- CASE statements
- Advanced SQL querying
- JOIN operations
- Business insights generation

## Key Insights

Some important findings from the analysis:

- Foreign transactions showed higher fraud rates
- Low device trust scores were strongly linked to fraud
- High transaction velocity increased fraud probability
- Certain merchant categories had significantly higher fraud rates
- High-risk combinations improved fraud detection accuracy
